# Makeup Guide Website 💄🌸

Welcome to the **Makeup Guide Website**, a complete step-by-step tutorial for applying makeup products—from primer to setting spray. Each product has a clear image and usage guide, all in a visually pleasing, animated floral theme.

## Features

- Detailed makeup tutorial with all essential products:
  - Primer, Foundation, Concealer, Powder
  - Eyeshadow, Eyeliner, Mascara, Blush, Bronzer
  - Lipstick, Setting Spray
- Step-by-step guidance for correct application
- Beautiful animated background with sparkling floral effects
- Mobile and desktop friendly layout

## Files

- `index.html` — Main page with all makeup tutorials
- `style.css` — Styles for the website
- `script.js` — Animations and interactive effects
- `images/` — Product images and animated shimmer

## How to Use

1. Clone or download this repository.
2. Open `index.html` in any web browser.
3. Explore the tutorials and enjoy the animated background!
